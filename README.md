Upload the new modlist in here.
It has to be named mods.html.
The mods.html file can be generated by exporting the modlist via Arma 3 launcher:
1) go to "MODS"
2) have all the mods that you want to export selected
3) click on "More"
4) select "Export list of mods to a file"
5) Select "Only loaded mods"
6) Name the file "mods.html"
7) Upload it to this repository
8) PROFIT!
